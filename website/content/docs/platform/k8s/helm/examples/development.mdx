# Development

~> **Important Note:** This chart is not compatible with Helm 2. Please use Helm 3.6+ with this chart.

The below `values.yaml` or Terraform configuration can be used to set up a single development Vault server.

<CodeTabs>

<CodeBlockConfig>

```yaml
# values.yaml
server:
  dev:
    enabled: true
```

</CodeBlockConfig>

<CodeBlockConfig>

```hcl
resource "helm_release" "vault" {
  name       = "vault"
  repository = "https://helm.releases.hashicorp.com"
  chart      = "vault"

  set {
    name  = "server.dev.enabled"
    value = "true"
  }
}
```

</CodeBlockConfig>

</CodeTabs>
